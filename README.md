# portfolio

Personal portfolio site for Jamison Kimminau — built as both a destination for recruiters and a deliberate showcase of motion design and frontend craft.

## Status

Pre-development. Scope and design language defined; implementation in progress.

## Why this exists

A portfolio site is two things: a landing pad for recruiters, and a demo of what its author is capable of. This one leans into the second purpose — using motion, typography, and pacing as the primary medium, with case studies that read more like long-form essays than bullet lists.

## Planned stack

| Layer | Choice | Rationale |
|---|---|---|
| Framework | Next.js 15 (App Router) | SSG/ISR for fast first paint |
| Motion | Framer Motion + GSAP ScrollTrigger | UI motion + scroll choreography |
| Content | MDX | Long-form case studies in source |
| Typography | Variable fonts | Distinct visual identity |
| Deploy | Vercel | Standard |

## Planned content

- Landing with motion-driven hero
- Case studies (long-form) for each major project
- Writing / notes section
- Contact + resume
